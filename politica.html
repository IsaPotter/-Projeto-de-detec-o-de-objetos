Sua Privacidade é Prioridade

A funcionalidade principal de detecção de objetos deste site é 100% anônima. Para usuários que criam uma conta, coletamos apenas os dados essenciais para o login.

1. Coleta de Dados

Nossa política de coleta de dados é dividida em duas partes:

Para todos os visitantes (uso anônimo):

  • A detecção de objetos via webcam é processada 100% no seu navegador.
  • Nenhuma imagem ou vídeo é enviado ou armazenado em nossos servidores.

Para usuários que criam uma conta:

  • Email: Coletamos seu endereço de email para ser usado como identificador único de login.
  • Senha: Sua senha é armazenada de forma segura usando criptografia forte (hashing). Nós nunca temos acesso à sua senha original.

2. Uso da Webcam

O acesso à webcam é solicitado apenas para realizar a detecção de objetos em tempo real.

O processamento é local: Todas as imagens capturadas são processadas diretamente no seu navegador usando TensorFlow.js

Nada é gravado: As imagens não são salvas em disco ou enviadas para qualquer servidor

Você tem controle: Pode desativar a webcam a qualquer momento e revogar permissões nas configurações do navegador

3. Cookies

Este site não utiliza cookies de rastreamento.

4. Links Externos

Este site pode conter links para redes sociais e outros sites externos. Não nos responsabilizamos pelas práticas de privacidade desses sites. Recomendamos que você leia as políticas de privacidade de qualquer site que visitar.

5. Segurança dos Dados da Conta

Levamos a segurança dos seus dados a sério. As senhas dos usuários são protegidas com a tecnologia de hashing bcrypt, o que significa que mesmo em caso de uma violação de dados, sua senha original não pode ser recuperada.

6. Seus Direitos

Você tem total controle sobre o uso deste site:

  • Pode negar o acesso à webcam a qualquer momento
  • Pode revogar permissões nas configurações do navegador
  • Pode optar por não criar uma conta e usar a funcionalidade principal de forma anônima.

7. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. Recomendamos revisar esta página periodicamente para estar ciente de quaisquer alterações.

Dúvidas?

Se tiver dúvidas sobre esta política de privacidade, entre em contato através das redes sociais disponíveis no rodapé do site.
